# @durkdotdev/aria-types

Type definitions for WAI-ARIA Accessibility.

## Installation

```bash
npm install @durkdotdev/aria-types -D
# or
yarn add @durkdotdev/aria-types -D
# or
pnpm add @durkdotdev/aria-types -D
```

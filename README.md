# @durkdotdev/aria-typed

Type definitions for WAI-ARIA Accessibility.

## Installation

```bash
npm install @durkdotdev/aria-typed -D
# or
yarn add @durkdotdev/aria-typed -D
# or
pnpm add @durkdotdev/aria-typed -D

```
